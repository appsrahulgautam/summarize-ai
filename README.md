## üåà Project Overview

A high-performance, full-stack AI SaaS platform designed for speed and scalability. This application enables users to upload PDF documents and receive **instant, structured summaries** powered by a hybrid AI engine.

### üéØ Key Highlights

- **ü§ñ Smart AI Fallback:** Dynamic routing between **OpenAI (GPT)** and **Google Gemini** for 100% reliability.
- **üí≥ Pro Monetization:** Full **Stripe Integration** with subscription tiers and automated billing portals.
- **üîí Secure Infrastructure:** Enterprise-grade file handling via **UploadThing** and type-safe data with **Drizzle ORM**.
- **‚ö° Modern Stack:** Built on **Next.js 14 App Router** for lightning-fast server-side rendering.
